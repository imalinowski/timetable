Manual TimeTable
----- Функционал UI -----

1. LogIn / Registration
Чтобы войти приложение (Зарегистрироваться, создастся новый аккаунт), нужно нажать на кнопку LogIn на странице Home ("/")

2. Редактирование группы
Нажать кнопку Edit (переход к "/edit"), выбрать из выпадающего списка доступную группу

3. Посмотреть расписание (перейти к "/timetable")
Нажать на кнопку ShowTimeTable на странице Home ("/")

4. Записаться на событие (лекцию/занятие)
4.1 На странице "/timetable" нажать AllEvents (переход к просмотру всех событий "/events")
4.2 из списка выбрать доступное и нажать Enroll

(Ограничение - возможна ошибка при записи, если в это время пользователь занят)

5. Выйти из события (лекции/занятия)
5.1 На странице "/timetable" нажать AllEvents (переход к просмотру всех событий "/events")
5.2 из списка выбрать доступное и нажать Leave

----- Teacher role -----

6 Создать событие (лекцию/занятие)
6.1 Перейти на страницу всех событий "/events"
6.2 Если уровень доступа 'Teacher', будет кнопка Create Event. Нажать ее. Выполнится переход к "/edit"
6.3 Заполнить имя, день недели, время и место. Нажать create

----- Функционал не UI -----

7. Список всех user - GET "/user"

8. Новая группа - POST "/group" body - JSON (id:int, name: string)

9. Новая локация - POST "/location" body - JSON (name: string, coordinates: string)
